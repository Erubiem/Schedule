# 캐릭터 제작 플랜 (가안)
## 1. 캐릭터 이미지 제작
1) 기간 : 장당 2~3일 -> 주당 2장 정도 
2) 방향 : 뿌요 이미지를 쓰느냐에 따라 달라진다. 
    - 사용할 경우 -> 이질감 없이 추가 캐릭터 이미지 사용할수 있게 작업
    - 사용할수 없을 경우 -> 기존 작업물과 여러 스타일을 참조 
3) 총량 : 개발 기간내에 총량
    - 잔여 개발 예상 기간은 2달로 예싱
    - 해당 기간에 추가 되는 이미지는 20여장 미만이 될것으로 예상
    - 설정과 디자인이 변수
